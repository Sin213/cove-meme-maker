# Starter Templates

These four PNG files are placeholder backgrounds shipped with Cove Meme Maker to
demonstrate the tab-web template gallery. They are intentionally simple so the
feature works out of the box without requiring external assets.

| File | Description |
|------|-------------|
| `blank-white.png` | Solid white — 800 × 600 |
| `blank-black.png` | Solid black — 800 × 600 |
| `blank-gray.png`  | Mid-gray (#808080) — 800 × 600 |
| `gradient-dark.png` | Dark blue-gray vertical gradient — 800 × 600 |

## Adding real meme templates

Drop any PNG/JPG/WebP image into this directory. The filename stem becomes the
template ID (lowercase, non-alphanumeric characters collapsed to `-`). The gallery
refreshes on each app launch — no code changes needed.

Example: `drake-pointing.png` → ID `drake-pointing`, shown as "Drake Pointing".
